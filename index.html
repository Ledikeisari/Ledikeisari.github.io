RunQuest

2021 | Project Manager

RunQuest is a work in progress mobile game I am developing with a fellow student. The game aims to provide motivation to move for gamers by providing daily step goals to reach and interesting gameplay.

I am the project manager of this game and I have created and implemented all the art assets for this project. I have also programmed and visualised a fully functional 1v1 turn based combat system that has skills, affinities and other basic rpg elements with Unity3D and C#. I also implemented a working stepmeter that uses the pedometer found in almost every phone available.

OffBeat

2021 | Project Manager

OffBeat is a four student game project done for a school game developement course. The game is a music based jrpg that has evolving music during the combats.

I was the project manager of this game and on top of that I created and implemented all the character models, animations, UI elements and programmed a fully functional jrpg turn based combat system with many enemies and player characters with skills and equipment. I also composed 12 songs for the game which some evolve based on the decisions made by the player during the combat

SnowDrop

2020 | Project Manager

SnowDrop is a four student project made for a game development course. It's an Asia-themed metroidvania that has one complete stage and a few enemies. The game also has multiple weapons to use and a leveling system as well as stats that grow with those levels.

My role in this project was to be the project manager. But on top of that I created all the 3D models for the game from scracth, Animated and implemented the models to Unity3D and programmed some behaviours for the enemies and the main character. I also created all the UI elements Including the HUD, Buttons and Menus, aswell as the Title image.
